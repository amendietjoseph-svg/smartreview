Questions fréquentes

Allez dans la section Comptes et cliquez sur le bouton "Nouveau Compte". Remplissez les informations requises (nom, type de compte, solde initial, etc.) et sauvegardez. Vous pouvez ajouter plusieurs comptes personnels et/ou de prop firms.
Le Trading Score™ est une métrique exclusive de SmartFX-Review qui évalue la qualité globale de votre trading sur une échelle de 0 à 100. Il prend en compte plusieurs facteurs : le win rate, le risk-reward moyen, la gestion du risque, la cohérence psychologique et la discipline de suivi de journal.
L'IA Coach utilise Claude d'Anthropic pour analyser vos trades et vous fournir des insights personnalisés. Il peut détecter vos edges (combinaisons gagnantes), analyser votre psychologie de trading, générer des rapports quotidiens et hebdomadaires, et vous donner des recommandations d'amélioration. Pour l'utiliser, vous devez configurer votre clé API Anthropic dans les Paramètres.
Actuellement, l'import automatique n'est pas disponible. Vous devez entrer vos trades manuellement via le Journal. Nous travaillons sur des intégrations avec les principaux brokers pour permettre l'import automatique dans le futur.
Oui, toutes vos données sont stockées localement dans une base de données SQLite sur votre machine. Aucune donnée n'est envoyée à des serveurs externes, sauf pour l'analyse par l'IA Coach (avec votre consentement via la clé API). Vous avez un contrôle total sur vos données.
Allez dans les Paramètres > Notifications. Vous pouvez activer les alertes de risque qui vous avertiront quand vous approchez ou dépassez votre limite de risque quotidien. Vous pouvez également configurer des rappels pour compléter votre journal de trading.
Le Calendrier de Performance vous permet de visualiser vos résultats jour par jour sous forme de calendrier. Chaque jour affiche votre P&L, votre nombre de trades et votre win rate. Vous pouvez également voir une heatmap horaire pour identifier vos meilleures heures de trading, ainsi que vos meilleures et pires journées.
Oui, vous pouvez exporter vos statistiques en format CSV depuis la page Statistiques en cliquant sur le bouton "Exporter". Cela vous permet d'analyser vos données dans Excel ou d'autres outils.
